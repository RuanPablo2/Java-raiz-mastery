# ☕ Java Raiz: Road to Oracle Certified Professional (OCP)

![Java](https://img.shields.io/badge/Java-21-blue) ![Certification](https://img.shields.io/badge/Exam-OCP%20Java%20SE%2021-orange) ![Status](https://img.shields.io/badge/Status-Em%20Progresso-brightgreen)

Repositório focado em dominar a base da linguagem Java, eliminando a dependência de "mágicas" de frameworks e pavimentando o caminho para a certificação **Oracle Certified Professional**. Aqui, o foco é entender o "porquê" de cada linha de código.

## 🎯 Objetivos do Repositório

- **Fortalecimento da Lógica:** Exercitar algoritmos e manipulação de dados sem bibliotecas prontas.
- **Domínio da OO:** Aplicar Encapsulamento, Herança e Polimorfismo no nível exigido pela Oracle.
- **Preparação OCP:** Resolver desafios com foco em detalhes de compilação, escopo e memória.
- **Base para Especialista:** Construir o alicerce necessário para futuras arquiteturas de Microserviços e alta escala.

## 🛠 Como Executar

Cada exercício possui sua própria classe `main`.

1. Clone o repositório.
2. Acesse a pasta do módulo desejado.
3. Compile e rode:
   ```bash
   javac NomeDoExercicio.java
   java NomeDoExercicio
   ```
